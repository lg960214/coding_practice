## How To?

#### 1. 해결할 문제의 번호를 이름으로 한 폴더 생성 후 main에 push
#### 2. 최초 문제 해결자가 각 문제 폴더의 README.md에 세부설명 추가
#### 3. '(백준 문제번호)_(자신의 브랜치 이름).py' 형식으로 push
